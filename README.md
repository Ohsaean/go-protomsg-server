# gogpd

##### golang - google protocol buffer demo

install
```bash
$> go get github.com/ohsaean/gogpd
```
server
```bash
$> go build github.com/ohsaean/gogpd/server
```
server (debug mode)
```bash
$> go build -tags 'debug' github.com/ohsaean/gogpd/server
```
client
```bash
$> go build github.com/ohsaean/gogpd/client
```
